# Athana SDK for Cocos

This plugin is for the Cocos Creator platform, with a minimum support of Cocos Creator 3.4.2.

## Installation

Download methods:

- [Git Repo Clone](https://github.com/AthanaSDK/athana_cocos_plugin.git)

- [Download ZIP](https://github.com/AthanaSDK/athana_cocos_plugin/releases)

1. Import to the Cocos Create project:

Open Extension Manager (Extension -> Extension Manager)

- Import directory: `Import Extension Folder`

- Import ZIP package: `Import Extension File`

2. Navigate to the project plugin directory:

{PROJECT_ROOT}/extension/athana-cocos

3. Compile the plugin

```shell

# Execute commands

npm install
npm run build
```
