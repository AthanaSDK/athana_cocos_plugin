# Athana SDK for Cocos

此插件适用于 Cocos Creator 平台，最低支持 Cocos Creator 3.4.2

## 安装

下载方式：
- [Git Repo Clone](https://github.com/AthanaSDK/athana_cocos_plugin.git)
- [Download ZIP](https://github.com/AthanaSDK/athana_cocos_plugin/releases)

1. 导入至 Cocos Create 项目：

        打开 Extension Manger （Extension -> Extension Manager）
        - 目录导入： `Import Extension Folder`
        - ZIP包导入：`Import Extension File`

2. 进入项目插件目录：

        {PROJECT_ROOT}/extension/athana-cocos

3. 编译插件
        
```shell
# 执行命令
npm install
npm run build
```
